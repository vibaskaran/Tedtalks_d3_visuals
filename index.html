Process

We first had to retrieve our dataset to work with. This was done by accessing Kaggle and selecting the TED talks data set.

  • Retrieve Kaggle dataset.
  • Scrape TED website for TED speaker images.
  • Analyze the data set between two topics Rating Categories & Top Speakers.
  • Finally use D3 and Dimple JS library to create the charts.

API

We created an API to access different categories of TED talks which then can be used for our D3 Charts.
